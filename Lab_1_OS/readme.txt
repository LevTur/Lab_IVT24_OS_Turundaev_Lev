Простая оболочка командной строки myShell

Использование:
./myshell          # Интерактивный режим
./myshell batch.txt # Пакетный режим (команды берутся из файла)

Поддерживаемые команды:
cd [каталог]       - сменить рабочий каталог
clr                - очистить экран
dir [каталог]      - вывести содержимое каталога
environ            - вывести переменные окружения
echo <сообщение>   - вывести сообщение
help               - вывести справку
pause              - дождаться нажатия Enter
quit               - завершить оболочку

Перечисленные команды поддерживают перенаправление ввода-вывода:
command < infile > outfile
command << infile >> outfile

Фоновый режим:
command &

Примеры:
ls -la > out.txt           # Вывести результат ls в файл out.txt
sort < data.txt            # Чтение данных из файла data.txt
find . -name *.txt &       # Найти файлы в фоновом режиме
